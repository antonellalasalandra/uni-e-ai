# UNI E AI

UNI E AI è un percorso parallelo alla laurea in Ingegneria Informatica.

L'obiettivo finale è costruire da zero un piccolo modello linguistico autoregressivo, comprendendo la matematica, gli algoritmi, il codice e le scelte architetturali alla base dei moderni sistemi di intelligenza artificiale.

## Obiettivi

Il progetto ha due obiettivi principali:

1. superare gli esami di Ingegneria Informatica comprendendo realmente gli argomenti;
2. trasformare ogni esame in un componente concreto utile alla costruzione di un piccolo modello AI.

## Metodo

Ogni argomento viene affrontato attraverso cinque livelli:

1. intuizione;
2. definizione formale;
3. esempio matematico;
4. implementazione in Python;
5. collegamento con l'intelligenza artificiale.

Per ogni esame vengono prodotti:

- teoria;
- esercizi universitari;
- esercizi pratici in Python;
- test automatici;
- un progetto progressivo;
- una relazione finale;
- un aggiornamento delle competenze acquisite.
